# Kubernetes Learning Repo

This repo is created to help understand Kubernetes architecture and startup concepts.  
It includes notes, diagrams, and practical examples.

---

## Files

- `kubernetes_architecture.md` – Overview of Kubernetes architecture components.
- `kubernetes_architecture_startup.md` – Examples and commands for getting started with Kubernetes.
- `images/kubernetes_architecture_diagram.png` – Architecture diagram (optional local copy).

---

## Usage

1. Read `kubernetes_architecture.md` first.  
2. Try the examples in `kubernetes_architecture_startup.md`.  
3. View the diagram in the `images/` folder if needed.
